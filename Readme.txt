В папке data/ лежат данные для 9-и тестовых гамма-всплесков 
и скрипты для их отображения.

Исходные данные (в папке data/)
-------------------------------
Исходные данные прредставляют собой запись числа отсчётов детектора во воременных 
интервалах (бинах) с преременной длительностью в 12 диапазонах энергий фотонов.

Расшивровка названия файла с данными
krf20090605_74449_1_S1.thr
   ^дата    ^время  ^детектор
'_1_' в имени файла соответствует номеру анализатора в детекторе, 
здесь рассматриваются только данные 1-го анализатора, 
который записывает данные о квантах с энергиями < 1000 кэВ.

Список всплесков - krf_grbs_for_test.txt
В этом файле стоит обратить внимание на колонку
Source - тип источника, если там указана дата, то это гамма-всплеск из удалённой галактики, 
если SGR (soft gamma-repeater) или AXP (Anomalous X-ray pulsar), то это всплеск от нейтронной звезды в галактике.


Скрипты
-------

plot_krf_all.py
- Основной скрипт. На вход получает таблицу с параметрами всплесков krf_grbs_for_test.txt,
  в результате создаёт данные с загрублённым временным разрешением для удобства отображения
  (можно делать разрешение 2, 8, 16, 32, 64 и 256 мс, при этом длина создаваемых данных 
  будет ограничена интервалом с разрешением меньше требуемого)

krf_rebin.py
- Скрипт для создания данные с загрублённым временным разрешением.

plot_krf_lc.py
- Скрипт для создания рисунков.

Задача
------
1. Для каждого всплеска каждом энергетическом канале, выделить интервал фона, на котором его можно считать постоянным.
2. С использованием полученного фона, провести поиск начала и конца всплеска в данных.
